# VBA service #

## Features
* Validate incoming requests
* Create / update requests
* Retrieve requests

## Development
#### Install dependencies
```bash
npm install
cd src && npm install
```
#### Run app 
```bash
npm run dev
```
#### or watch changes
```bash
npm run watch
```

## Production
#### Install dependencies
```bash
npm install
```
#### Build dist
```bash
npm run build
```
#### Run
```bash
npm run start
```
